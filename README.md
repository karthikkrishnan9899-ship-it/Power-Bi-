# PowerBI-Assignment-1---Data-Transformation-Data-Modeling

E-Commerce Sales Data Analysis using Power BI

## Project Overview

This project was developed using Power BI and Power Query Editor on an E-commerce sales dataset. The objective of the assignment was to perform data import, transformation, data cleaning, aggregation, and relationship modeling.

## Datasets Used

* List of Orders
* Order Details
* Sales Target

## Tasks Performed

* Data Import using Power BI
* Data Transformation using Power Query Editor
* Restricting rows and changing data types
* Text formatting and column creation
* Profit Margin and Profit Status calculation
* Merging datasets using Order ID
* Handling missing and duplicate data
* Sorting and filtering data
* Grouping and aggregation
* Data modeling and relationship creation

## Tools Used

* Microsoft Power BI
* Power Query Editor

## Key Learning Outcomes

* Data cleaning and preparation
* Data transformation
* Aggregation techniques
* Relationship modeling
* Business data analysis
